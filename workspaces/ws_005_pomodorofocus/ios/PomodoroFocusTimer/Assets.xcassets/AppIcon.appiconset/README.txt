App Icon
========

Place a single 1024x1024 PNG named "AppIcon-1024.png" in this folder before
archiving. Xcode 16 generates all other required sizes automatically from the
single 1024 source.

The master artwork is provided in the repository at:
    shared/app_icon.svg

Export it to a 1024x1024 PNG (no alpha, sRGB) and save it here as
"AppIcon-1024.png".
